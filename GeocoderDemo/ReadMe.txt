### GeocoderDemo ###

===========================================================================
DESCRIPTION:

Demonstrates basic forward string geocodes and reverse coordinate geocodes. 
Also includes a distance between two placemarks calculator.

For a simple example of using forward geocoding using NSDictionary, refer to the code snippet inside
ForwardViewController's "viewDidLoad" method.

===========================================================================
BUILD REQUIREMENTS:

Xcode 4.2 or later, iOS 5.0 SDK or later.

===========================================================================
RUNTIME REQUIREMENTS:

iOS 5.0 or later.
